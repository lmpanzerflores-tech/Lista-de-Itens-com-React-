# Lista de Itens com React

Projeto desenvolvido com React para gerenciar uma lista de tarefas. A aplicacao permite criar tarefas com titulo e descricao, marcar itens como concluidos, remover tarefas e visualizar os detalhes de cada item.

## Funcionalidades

- Adicionar uma nova tarefa
- Informar titulo e descricao da tarefa
- Marcar tarefa como concluida
- Remover tarefa da lista
- Visualizar detalhes da tarefa em outra pagina
- Salvar as tarefas no localStorage do navegador

## Tecnologias utilizadas

- React
- Vite
- Tailwind CSS
- React Router DOM
- Lucide React
- UUID

## Como rodar o projeto

Clone o repositorio:

```bash
git clone https://github.com/lmpanzerflores-tech/Lista-de-Itens-com-React-.git
```

Acesse a pasta do projeto:

```bash
cd Lista-de-Itens-com-React-
```

Instale as dependencias:

```bash
npm install
```

Inicie o servidor de desenvolvimento:

```bash
npm run dev
```

Depois acesse no navegador:

```txt
http://localhost:5173
```

Este projeto foi criado para praticar conceitos fundamentais do React, como componentes, propriedades, estado, eventos, listas, rotas e persistencia de dados no navegador.
